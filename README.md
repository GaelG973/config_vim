## My vim config

## Install instructions

* cd ~
* git clone https://github.com/GlaconBleu/config_vim.git ~/.config_vim
* ln -s .config_vim .vim
* git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
* start vim
* type :VundleInstall
* restart vim

That's it ! Congratulations !
